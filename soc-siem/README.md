# 🔐 SOC / SIEM Lab - Incident Detection & Response

## 🇧🇷 Descrição

Este módulo simula cenários de segurança com foco em análise de logs e resposta a incidentes utilizando Wazuh.

---

## 🚨 Cenários simulados

- Brute force (falhas de login)
- Execução suspeita (PowerShell)
- Login fora de padrão
- Criação de usuário

---

## 🔎 Análise

- EventID 4624 / 4625
- Correlação de eventos
- Identificação de padrões anômalos

---

## 🛡️ Resposta

- Contenção de acesso suspeito
- Preservação de evidências
- Análise antes de ação

---

## 🧠 Objetivo

Simular o comportamento de um analista SOC em cenários reais.
